# Super Mario Bros. C++ Clone

A clone of Super Mario Bros. developed using C++ and raylib. I attempted to recreate the game's first level as accurately as possible.

## Features
- Power-Ups: Super Mushroom, Fire Flower, Super Star
- Enemies: Goombas and Koopas
- Bonus area accessible via pipe
- Iconic Music
- Converted and adapted from 4:3 to 16:9 aspect ratio

## Prerequisites

- A C++ compiler such as g++ is installed and added to PATH
- mingw32-make is installed and added to PATH![Uploading Screenshot 2026-07-17 174300.png…]()


## How to Run

Clone the repository
```
git clone https://github.com/idris-p/mario-bros-clone.git
cd mario-bros-clone
```

On Windows, run the build file:
```
.\build-MinGW-W64.bat
```

Run the executable:
```
.\bin\mario-bros-clone.exe
```
